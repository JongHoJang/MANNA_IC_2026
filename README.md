# MANNA IC 2026

MANNA IC 2026 컨퍼런스를 위한 웹앱입니다.

## 개요

- 로그인 후 홈, 시간표, 내 강의, 안내를 확인할 수 있습니다.
- 선택한 강의와 일정, 공지사항, 교통 및 숙소 안내를 한 화면에서 볼 수 있습니다.
- 모바일 사용성을 우선으로 구성했습니다.

## 주요 기능

- 참가자 로그인 및 입장
- 홈 화면 캐러셀 및 행사 소개
- 시간표 확인 및 선택세션 강의 확인
- 내 강의 확인
- 셔틀, 도보, 자차, 숙소 안내
- FAQ 확인

## 실행

```bash
npm install
npm run dev
```

## 접속 경로

- `/` 사용자 화면
- `/admin` 관리자 화면

## 기술 스택

- Next.js
- React
- TypeScript
- Supabase
